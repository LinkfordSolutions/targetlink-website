# TargetLink Brand Identity

## Brand Colors

### Primary Colors
- **Primary Blue**: `#0066FF` - Main brand color, represents trust and innovation
- **Secondary Cyan**: `#00D4FF` - Energy and modernity
- **Accent Red**: `#FF3D71` - Call-to-action and highlights
- **Purple**: `#7B61FF` - Creativity and premium feel

### Supporting Colors
- **Dark**: `#0A0E27` - Text and backgrounds
- **Light**: `#F5F7FA` - Backgrounds and cards
- **Success**: `#00E676` - Positive actions
- **Warning**: `#FFB800` - Alerts and notifications

## Gradients

### Primary Gradient
```css
linear-gradient(135deg, #0066FF 0%, #00D4FF 100%)
```
Used for: Main CTAs, headers, primary buttons

### Alternative Gradient
```css
linear-gradient(135deg, #7B61FF 0%, #FF3D71 100%)
```
Used for: Secondary CTAs, badges, highlights

### Full Spectrum Gradient
```css
linear-gradient(135deg, #0066FF 0%, #7B61FF 50%, #FF3D71 100%)
```
Used for: Hero sections, special elements

## Logo Design

### Concept
The TargetLink logo combines two key elements:
1. **Target**: Concentric circles representing precision and focus
2. **Link**: Chain elements in corners representing connectivity

### Logo Variations

#### Primary Logo (Header)
- Icon + Text combination
- Size: 40x40px icon
- Colors: Brand gradient (Blue → Purple → Red)
- Usage: Main navigation, header

#### Hero Logo
- Larger icon with glass morphism effect
- Background: White/10 with backdrop blur
- Border: White/20
- Usage: Hero section branding

#### CTA Logo
- Emphasized version with warm gradient
- Colors: White → Yellow → Red
- Size: 48x48px
- Usage: Call-to-action sections

#### Footer Logo
- Icon + Text combination
- Size: 32x32px icon
- Usage: Footer branding

### Logo Spacing
- Minimum clear space: 8px around logo
- Icon-to-text spacing: 8px (0.5rem)

## Typography

### Headings
- **Font**: Sora
- **Weights**: 600 (Semibold), 700 (Bold), 800 (Extrabold)

### Body Text
- **Font**: Inter
- **Weights**: 400 (Regular), 500 (Medium), 600 (Semibold)

## Usage Guidelines

### Buttons
- Primary: Blue gradient with white text
- Secondary: White/transparent with blue border
- Hover: Increase shadow and scale slightly

### Cards
- Background: White with subtle blue shadow
- Border: Light gray with blue accent on hover
- Hover: Lift effect with increased shadow

### Icons
- Background: Blue gradient
- Color: White
- Shape: Rounded squares (border-radius: 16px)

### Badges
- Background: Gradient (primary or alternative)
- Text: White, uppercase, bold
- Shape: Fully rounded (border-radius: 9999px)

## Accessibility

- All color combinations meet WCAG AA standards
- Minimum contrast ratio: 4.5:1 for normal text
- Minimum contrast ratio: 3:1 for large text
